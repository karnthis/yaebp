# README
## Getting Started

### Git Work
* git clone https://github.com/karnthis/yaebp.git
* git remote remove origin
* git remote add *NAME* *REMOTE-URL* (if desired)

### Shell Work
* cp sample.env .env
* cp sample.CHANGELOG.md CHANGELOG.md
* rm sample.CHANGELOG.md

### Project Start
* Populate .env
* Update fields in package.json:
	* name
	* description
	* version
	* keywords
	* author
* Run npm install
* Commit to git
* Start work
